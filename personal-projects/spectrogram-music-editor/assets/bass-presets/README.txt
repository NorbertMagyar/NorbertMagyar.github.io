Bundled bass preset imports
===========================

These MIDI files were copied from Wikimedia Commons for use as bundled bass-pattern
imports inside the spectrogram music editor.

Files and sources
-----------------

1. Walkingbass twelve-bar blues variation.mid
   Source:
   https://commons.wikimedia.org/wiki/File:Walkingbass.png
   Note on source page: "own work, public domain"

2. Texas & Pacific boogie woogie bass.mid
   Source:
   https://commons.wikimedia.org/wiki/File:%22Texas_%26_Pacific%22_boogie_woogie_bass.png
   Note on source page: released into the public domain by Hyacinth

3. Blues mi harmonie elementaire.midi
   Source:
   https://commons.wikimedia.org/wiki/File:Blues_mi_harmonie_elementaire.midi
   License on source page: CC BY-SA 4.0
   Author: Cdang

4. Didos Lament ground bass.mid
   Source:
   https://commons.wikimedia.org/wiki/File:Dido%27s_Lament_ground_bass.png
   Public Domain Mark / PD-old on source page

5. Bachs Crucifixus ground bass.mid
   Source:
   https://commons.wikimedia.org/wiki/File:Bach%27s_Crucifixus_B_Minor_Mass_ground_bass.png
   Public Domain Mark / PD-old on source page

The editor uses these files as bundled preset imports rather than redistributing them
as a standalone MIDI pack.
